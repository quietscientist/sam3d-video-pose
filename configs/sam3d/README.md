# SAM3D Configuration Files

Configuration files for SAM3 video segmentation and SAM 3D Body reconstruction.

## Available Configs

- `default.yaml` - Default configuration with all parameters
- `quick_test.yaml` - Fast testing configuration
- `batch_folder.yaml` - Batch processing configuration

## Usage

```bash
python scripts/process_video.py --config configs/sam3d/default.yaml
```
